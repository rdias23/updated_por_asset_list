Projects & Features

...by Ray Dias
Abstract: Herein described are various projects, and features, implemented by Ray Dias, between May 2014 and the present. The most recent projects described herein are "GYDR Finder" and "Tour Availabilities". The single biggest project by far is Travelogues or, more precisely, the Interactive Itinerary Authoring Tool used to create Travelogues. Code Links are samples only, and do not represent all relevant code.

Project code is in a private repository; consequently, all relevant code snippets have been posted via gist links.

Pictoura.com was originally a photo-sharing website (Rails 3.2) aimed at Travel and Photography Enthusiasts and Professionals. Currently it is being rennovated to function as a service that enables laymen and professionals to market themselves as Tour Guides. These Tour Guides are called GYDRs.

A recent development build of the GYDR Project can be accessed at the Staging URL: http://pictoura-staging.herokuapp.com If you want to look at anything besides the Home Page, you'll be prompted for credentials: type in the username 'pictoura' and the password 'pictoura1111' (the quote marks are not to be included).

Travelogues

Project Description: A sophisticated Single Page Application built with Backbone and Marionette. Used to create Interactive Itineraries (i.e. Travelogues) for Travellers, Vacation Package Vendors, or Tour Guides utilizing the GYDR service. Each Interactive Itinerary is a set of geo-tagged Travelogue Posts (i.e. Tour Stops) accompanied by descriptive text and a set of geo-tagged images (added via a "drag" and "drop" interface). Images may be drawn from the World Gallery, user created Galleries, of uploaded on the spot.
